# Keyword alignment (honest)

- Mirror job-description phrasing only when factually equivalent to `resume.md`.
- Target ~1–2 aligned terms per bullet; never keyword-stuff footers or hidden repeats.
- Use `job-language-mapping.md` to distinguish direct matches from bridges. Example: "plan status" may bridge toward entitlements, but it is not the same as entitlement enforcement unless `resume.md` says so.
- Expand acronyms once if the job description uses the long form and resume has the skill.
- If the job asks for a skill absent from the base resume -> **Do not claim**, not the tailored resume.
