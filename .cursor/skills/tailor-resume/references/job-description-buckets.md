# Job description buckets

Classify each job-description line as:

| Bucket | Use in `gaps.md` |
|--------|------------------|
| Must-have | What lines up, or What is missing |
| Preferred | What is missing, if relevant |
| Responsibilities | What lines up, when evidenced |
| Tools/stack | Skills alignment |
| Nice-to-have | What is missing, only if relevant |

Map rows to **What lines up**, **What is missing or not proven**, or **Do not claim**.
