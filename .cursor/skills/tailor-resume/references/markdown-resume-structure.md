# Markdown resume structure

1. Name + contact line
2. Summary (3–4 lines max for tailored version)
3. Work Experience (reverse chronological)
4. Skills (grouped if helpful)
5. Education

Job block template:

```markdown
### Title — Company (dates)

- Bullet with metric when available
- Bullet naming system, action, and outcome
- Bullet naming cross-functional or reliability impact
```

Use 4-6 bullets for recent roles when the master profile supports it. Older roles may stay shorter.

Optional mini-section under a role:

```markdown
**Notable work:** One sentence naming a project, systems touched, and rollout or user impact.
```

Group skills into 2-4 readable categories when the flat list becomes hard to scan.

Use HTML comments `<!-- anchor:company-role -->` only when needed for stable diffs.
