# Anti-patterns

Never:

- Invent employers, titles, dates, or tools
- Add skills from the job description that are not in `resume.md`
- Inflate seniority (e.g. "led" without leadership evidence)
- Hide keywords in white text or metadata
- Paste job-description paragraphs into the resume body
- Add unverifiable metrics
- Propose a suggested addition with no adjacent fact in `resume.md`
- Move a suggested-but-unverified claim into `tailored-resume.md`
- Write cover-letter hooks from facts that appear only in `suggested-additions.md`
- Treat related experience as a direct match when the source text is weaker
