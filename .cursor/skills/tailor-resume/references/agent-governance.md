# Agent governance

**Hierarchy:** `resume.md` > verified job-description facts > related experience > stylistic suggestions > unverified suggested additions

**Allowed:** rephrase, reorder, emphasize, trim length, align honest keywords, write check-before-using questions, frame honest gaps

**Forbidden:** new employers, new tools, new certifications, new degrees, unverified suggestion claims in `tailored-resume.md`, cover-letter facts sourced only from suggestions

**Suggestion gate:** `suggested-additions.md` may propose "if true" wording, but the user must verify and update `resume.md` before any suggestion becomes a resume fact.

**Human gate:** User reviews PR before applying; agent does not auto-submit applications.
